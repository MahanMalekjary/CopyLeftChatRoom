# CopyLeft Chat Room 🌍💬

An open-source, real-time chat room where users connect directly to a shared database and chat without centralized moderation.

## 🚀 Live Demo

Visit the online version:

https://mahanmalekjary.github.io/CopyLeftChatRoom/

Or simply download the project and run it locally. As long as you have an internet connection and access to the configured database, you can chat directly from localhost without needing the hosted website.

---

## ✨ Features

- ⚡ Real-time messaging
- 🌎 Global chat room
- 🖼️ Image uploads
- 🎥 Video uploads
- 📁 File uploads
- 🔄 Live synchronization
- 🖥️ Works from localhost
- 🎨 Fully customizable UI
- 🔧 Easy to modify source code
- 📂 Open-source
- ❌ No message monitoring
- ✅ Any type of photo, video, message, file is allowed

---

## 🛠 How It Works

Unlike traditional chat platforms, this project connects directly to the backend database.

This means:

- Users can clone or download the repository.
- Run the project locally.
- Connect to the shared database.
- Instantly participate in the chat room.

You are not required to use the hosted website. The GitHub Pages version is simply one way to access the chat.

## 📥 Installation

```bash
git clone https://github.com/MahanMalekjary/CopyLeftChatRoom.git
```

Open the project folder and launch it using your preferred local server.

After connecting to the configured database and with an active internet connection, you can start chatting immediately.
